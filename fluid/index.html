Boids

An implementation of the boids flocking algorithm using HTML canvas.

Each boid ("bird droid") is influenced by four forces: separation (avoid crashing with the close neighbors), alignment (align to the general direction of the flock), cohesion (stay close to the center of the flock) and obstacle avoidance (avoid crashing with the boxes).

You can modify the algorithm parameters using the sliders below and add new obstacles by clicking with the mouse on the canvas (source code).
