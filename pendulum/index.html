Kinematic Solvers

Implementation of several numeric integration solvers to model the kinematics of a spring, comparing their accuracy respect to the exact solution.

You can modify the time step of the solvers using the slider below, try increasing the time step to appreciate the differences between the methods (source code).


  • Exact solution
  • Euler method solver (wikipedia)
  • Midpoint method solver (wikipedia)
  • Euler-Cromer method solver (wikipedia)
  • Runge-Kutta 4th approximation solver(wikipedia)
